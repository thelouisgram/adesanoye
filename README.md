# A² Studio

Freelance client-conversion portfolio — websites and product UI built to book clients.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion

## Pages

- `/` — conversion home (hero, pitch, portfolio, services, about, CTA)
- `/apply` — project inquiry form (mailto)
